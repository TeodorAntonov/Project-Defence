# ProjectDefence - Fitness Administrative App

This application is a demo app for SoftUni Asp .Net/Web course 2022.

<h1>Resume:</h1>
Fintess App for any gym's administration. You have clients who want to have personal online trainers. They can apply for any registrated trainer in the system and if a trainer decide to train them, the clients will recieve a new fitness program with description who to perform the exerces. Tha system very basic level of quick actions for clients, trainers, and administrators.
